## About This Repository

This repository contains the complete computational reproducibility package and numerical data for the manuscript: *"A Scale-Cascading Delayed-Equilibration Framework for Nonequilibrium Complexity and Chaos"*. 

The theoretical framework unifies temporal and spatial chaos as frustrated equilibration across interacting scales, governed by a single dimensionless parameter, the Equilibration Number ($\mathcal{E}$). 

## Repository Structure

- `Code_Book.ipynb`: The core Jupyter Notebook that performs all numerical integrations and generates the figures presented in the paper. 
- `*.csv`: Output data files generated by the notebook containing time-series and phase-space data. These are used directly by `pgfplots` in the LaTeX manuscript to render vector graphics.
- `*.png`: Image outputs generated by the notebook for highly dense scatter plots and strange attractors.

## Numerical Methods

The Jupyter Notebook implements rigorous numerical methods to validate the theoretical claims:
- **Custom DDE Solver:** A high-precision fixed-step 4th-order Runge-Kutta solver with cubic Hermite interpolation is implemented from scratch to integrate Delay Differential Equations (DDEs).
- **Dynamical Systems:** Solves the dimensionless energy-balance models, Mackey-Glass equations, autonomous Lorenz multi-scale cascades, and non-autonomous Kuramoto phase models.
- **Robustness:** Implements strict transient discarding (e.g., ignoring the first $5 \times 10^5$ integration steps) to ensure the plotted data accurately represents the steady-state chaotic attractors.

## Usage & Reproducibility

To reproduce the numerical findings and regenerate all data/figures:

1. **Prerequisites:** Ensure you have Python installed with the following packages:
   ```bash
   pip install numpy scipy pandas matplotlib
   ```
2. **Run the Notebook:** Open `Reproduce_All_Figures.ipynb` in Jupyter and run all cells. 
3. **Output:** The notebook will re-integrate the dynamical systems and overwrite the `.csv` and `.png` files with fresh, verifiable outputs. Note that the 3-level Lorenz Cascade integration may take a couple of minutes due to the high-resolution time stepping required for chaotic accuracy.
